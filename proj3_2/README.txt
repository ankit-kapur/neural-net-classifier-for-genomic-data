
#### Instructions for running the IPython notebook (for DB-scan)

To run, you need to have the following:
1. Python 2.x [https://www.python.org/downloads/]
2. IPython installed [http://ipython.org/install.html].
3. The following python libraries: numpy, sklearn, matplotlib, networkx. You can install them with pip, for example 'sudo pip install numpy'

Now you simply need to run the 'ANN.ipynb' notebook, by writing this on the terminal: 

ipython notebook ANN.ipynb
